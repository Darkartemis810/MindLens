

# MindLens – AI Student Wellbeing Analyzer

## Overview
A polished web app that analyzes student mental wellbeing through AI-powered journaling, facial emotion detection, burnout prediction, and a mental health questionnaire — all presented on an insights dashboard.

---

## Pages & Features

### 1. Home / Landing Page
- Clean hero section explaining MindLens
- Privacy notice and "not medical advice" disclaimer
- Quick navigation to all features
- Login/signup with Supabase Auth (email-based)

### 2. Journal Entry Page
- Text area for daily journal writing
- On submit, sends text to a Supabase edge function that calls **Lovable AI (Gemini)** for:
  - Sentiment analysis (positive/negative/neutral)
  - Emotion detection (stress, anxiety, happiness, etc.)
  - Burnout keyword highlighting
  - Short AI insight (e.g., "Your journal indicates moderate academic stress")
- Displays mood score, emotion label, and insight
- Saves entry + analysis results to Supabase

### 3. Mood Camera Page
- Webcam access via browser using **face-api.js** (TensorFlow.js-based)
- Real-time face detection with emotion classification (happy, sad, neutral, angry, surprised, tired)
- Displays detected emotion on screen with a visual indicator
- Option to save a mood snapshot to the database

### 4. Burnout Check Page
- Form with lifestyle inputs: sleep hours, study hours, screen time, social interaction level, self-reported stress (1-10)
- Sends inputs to a Supabase edge function that uses **Lovable AI** to predict burnout risk (Low / Medium / High) based on the input pattern
- Displays risk level with color-coded badge and personalized tips
- Saves result to Supabase

### 5. Mental Health Questionnaire Page
- Short questionnaire (8-10 questions) inspired by standard wellbeing scales (PHQ/GAD style)
- Measures anxiety, stress, and emotional fatigue
- Calculates a wellbeing score client-side
- Score saved to Supabase for trend tracking

### 6. Insights Dashboard
- **Daily Mood** card (from journal + camera)
- **Burnout Risk** indicator (from burnout check)
- **Journal Sentiment** summary
- **Emotion Detection** latest result
- **Weekly Mood Trend Chart** using Recharts (line/bar chart over 7 days)
- **AI Suggestions** section with personalized tips (take a break, improve sleep, reduce screen time) generated by Lovable AI based on combined signals

### 7. AI Wellbeing Chatbot (Bonus)
- Simple chat interface for supportive conversation
- Powered by Lovable AI with a wellbeing-focused system prompt
- Streaming responses for a natural feel

---

## Tech Architecture

- **Frontend**: React + TypeScript + Tailwind CSS + shadcn/ui components
- **AI**: Lovable AI Gateway (Gemini) via Supabase edge functions for sentiment analysis, burnout prediction, suggestions, and chatbot
- **Face Detection**: face-api.js running in-browser
- **Database**: Supabase (journal entries, mood records, burnout results, questionnaire scores)
- **Auth**: Supabase Auth with user profiles
- **Charts**: Recharts for mood trends
